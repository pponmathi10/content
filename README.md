AI INTELLIGENT RESUME SCREENING SYSTEM
====================================

This project is an AI-powered resume screening and ranking system
built using Streamlit and NLP techniques.

FEATURES
--------
• Resume parsing (PDF)
• Job Description matching
• Candidate ranking
• Confidence level detection
• Experience estimation
• Final AI suitability verdict

TECHNOLOGY STACK
----------------
• Python
• Streamlit
• Scikit-learn
• TF-IDF
• Cosine Similarity
• NLP

HOW TO RUN LOCALLY
-----------------
1. Install dependencies:
   pip install -r requirements.txt

2. Run application:
   streamlit run app.py

DEPLOY ON STREAMLIT CLOUD
------------------------
1. Push files to GitHub
2. Go to https://streamlit.io/cloud
3. Select repository
4. Choose app.py
5. Click Deploy

OUTPUT
------
• Ranked candidates
• Match percentage
• Confidence score
• Experience score
• Final verdict (Highly Suitable / Suitable / Not Suitable)
